# ✅ **ARRAYS — 10 Exercícios Progressivos (usando `array`)**

### **1. Criando um array de inteiros**

Crie um array de inteiros (`'i'`) contendo cinco valores escolhidos pelo aluno e exiba o array completo.

---

### **2. Verificando o tipo dos itens**

Mostre o tipo do array e o tipo de um item individual, explicando a diferença para listas.

---

### **3. Acessando posições**

Exiba o primeiro, o terceiro e o último elemento do array usando índices positivos e negativos.

---

### **4. Alterando valores**

Permita que o usuário informe um índice e um novo valor. Atualize esse valor no array.

---

### **5. Adicionando valores**

Use `append()` para adicionar um novo número inteiro ao array.

---

### **6. Inserindo valores no meio**

Peça ao usuário uma posição e um valor para inserir usando `insert()`.

---

### **7. Removendo itens**

Peça um valor ao usuário e remova esse valor do array usando `remove()`.
Explique o erro caso o valor não exista.

---

### **8. Convertendo lista para array**

Peça ao usuário uma lista de números (entrada separada por espaço), converta para array e exiba o resultado.

---

### **9. Array de floats**

Crie um array do tipo `'f'` com cinco números decimais e calcule a média.

---

### **10. Comparação prática array vs lista**

Crie uma lista e um array com a mesma quantidade de elementos.
Exiba o tamanho em bytes de cada um usando:

* `array.buffer_info()`
* `sys.getsizeof()`

Mostre que arrays ocupam menos memória.
