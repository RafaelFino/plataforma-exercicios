# ✅ **DICIONÁRIOS — 10 Exercícios Progressivos**

### **1. Criando um dicionário simples**

Crie um dicionário chamado `aluno` com as chaves: `"nome"`, `"idade"` e `"curso"`. Preencha com valores fictícios.

---

### **2. Acessando valores**

Mostre o valor associado às chaves `"nome"` e `"curso"`.

---

### **3. Alterando valores**

Troque o valor da chave `"idade"` por um valor digitado pelo usuário.

---

### **4. Adicionando nova chave**

Adicione ao dicionário a chave `"nota_final"` com valor fornecido pelo usuário.

---

### **5. Removendo chave**

Remova a chave `"curso"` do dicionário usando `pop()`.

---

### **6. Percorrendo dicionário**

Mostre todas as chaves e valores do dicionário usando um `for` com `.items()`.

---

### **7. Dicionário de listas**

Crie um dicionário que contenha:

* `"nome"`: string
* `"notas"`: lista de 4 números digitados pelo usuário

Calcule a média das notas.

---

### **8. Dicionário de dicionários**

Crie um dicionário chamado `turma` onde cada chave é o nome de um aluno e o valor é outro dicionário com `"idade"` e `"nota"`.
Exiba a nota do aluno escolhido pelo usuário.

---

### **9. Contando frequência de palavras**

Peça ao usuário uma frase.
Conte quantas vezes cada palavra aparece usando um dicionário.
Exiba o resultado.

---

### **10. Sistema simples com dicionário**

Crie um sistema de cadastro de produtos:

* Cada produto tem: nome, preço e quantidade
* Permita adicionar produtos
* Permita listar produtos
* Permita atualizar quantidade
