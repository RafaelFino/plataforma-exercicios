# ✅ **LISTAS — 10 Exercícios Progressivos**

### **1. Criando sua primeira lista**

Crie uma lista chamada `frutas` contendo três frutas que você gosta. Em seguida, exiba toda a lista na tela.

---

### **2. Acessando elementos pelo índice**

Usando a lista `frutas` criada antes, mostre na tela:

* a primeira fruta
* a segunda fruta
* a última fruta (usando índice negativo)

---

### **3. Alterando um elemento da lista**

Peça ao usuário para digitar uma nova fruta e substitua o segundo elemento da lista `frutas` por essa fruta digitada.

---

### **4. Adicionando elementos ao final**

Peça ao usuário três novas frutas e adicione cada uma ao final da lista usando `append()`.

---

### **5. Inserindo elementos em posições específicas**

Insira a fruta `"uva"` na primeira posição da lista, sem substituir nenhum elemento (use `insert()`).

---

### **6. Removendo elementos da lista**

Peça ao usuário o nome de uma fruta e remova essa fruta da lista usando `remove()`.
Caso a fruta não exista, mostre uma mensagem explicando o erro.

---

### **7. Ordenando listas**

Crie uma lista com cinco números aleatórios escolhidos pelo usuário e exiba:

* a lista original
* a lista em ordem crescente
* a lista em ordem decrescente

---

### **8. Contando elementos**

Dada uma lista de nomes, peça ao usuário um nome e mostre quantas vezes esse nome aparece na lista usando `count()`.

---

### **9. Lista com loop**

Peça ao usuário 5 números e armazene-os em uma lista. Depois, percorra a lista com `for` e some todos os valores, exibindo o total.

---

### **10. Filtrando elementos**

Dada uma lista de números, crie uma nova lista contendo apenas os números pares.
Exiba a lista final.
