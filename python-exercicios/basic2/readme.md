# Exercicios Python
Este repositório contém uma coleção de exercícios práticos em Python, projetados para ajudar iniciantes e desenvolvedores intermediários a aprimorar suas habilidades de programação.

## Instruções
Para começar a trabalhar com os exercícios, siga os passos abaixo:
1. Certifique-se de ter o Python instalado em sua máquina. Você pode baixar a versão mais recente do Python em [python.org](https://www.python.org/downloads/).
2. Caso ainda não possua, crie uma conta no GitHub.com para hospedar seu repositório remotamente.
3. Crie um repositório no GitHub e clone-o para sua máquina local
4. Use o seu editor de códigos favorito (como VSCode, PyCharm, etc.) para abrir o diretório do repositório clonado.
5. Cada exercício deve ser salvo em um arquivo `.py` separado dentro do diretório do repositório.
6. Cada exercício deve conter comentários explicativos para facilitar o entendimento do código.
7. Cada exercício deve ser testado para garantir que funcione corretamente antes de ser submetido, cada arquivo deve ter um arquivo de teste seguindo o mesmo nome do arquivo do exercício acrescido de `_test` (por exemplo, `exercicio1.py` e `exercicio1_test.py`).
8. Após completar os exercícios, faça commit das suas alterações e envie-as para o repositório remoto no GitHub.

## Conteúdo
- 1. Exercícios de lógica de programação e matemática básica
    1. Números pares e ímpares: Peça para o usuário inserir um número e determine se ele é par ou ímpar.
    2. Calculadora simples: Crie uma calculadora que possa realizar operações básicas como adição, subtração, multiplicação e divisão.
    3. Fatorial: Calcule o fatorial de um número fornecido pelo usuário.
    4. Calculando a média: Peça ao usuário para inserir uma lista de números e calcule a média.
    5. Calcule a potência: Peça ao usuário para inserir uma base e um expoente, e calcule a potência.

- 2. Manipulação de strings
    1. Inversão de string: Peça ao usuário para inserir uma string e exiba a string invertida.
    2. Contagem de vogais: Conte o número de vogais em uma string fornecida pelo usuário.
    3. Palíndromo: Verifique se uma string é um palíndromo.
    4. Substituição de palavras: Peça ao usuário para inserir uma frase e substitua uma palavra específica por outra.
    5. Formatação de strings: Formate uma string para exibir um nome completo a partir de nome e sobrenome fornecidos pelo usuário.
    6. Dadas duas palavras, verifique se elas são anagramas uma da outra.

- 3. Manipulação de arquivos
    1. Leitura de arquivo: Leia o conteúdo de um arquivo de texto e exiba-o na tela.
    2. Escrita em arquivo: Peça ao usuário para inserir texto e salve-o em um arquivo.
    3. Contagem de linhas: Conte o número de linhas em um arquivo de texto.
    4. Cópia de arquivo: Crie uma cópia de um arquivo existente.
    5. Adicionar conteúdo a um arquivo: Peça ao usuário para inserir texto e adicione-o ao final de um arquivo existente.

## Dicas
- Comente seu código para explicar a lógica por trás de suas soluções.
- Teste seus exercícios com diferentes entradas para garantir que funcionem corretamente.
- Consulte a documentação oficial do Python em [docs.python.org](https://docs.python.org/3/) para obter ajuda sobre funções e bibliotecas.
- Divirta-se e aproveite o processo de aprendizado!
- Se não entender algo, PERGUNTE!
